🧹 STEP-BY-STEP DATA CLEANING IN EXCEL

Assumption: Your data is already properly split into columns like
Order_ID | Order_Date | Customer_Name | Region | Sales | Quantity | Discount

✔️ 1. Handle Missing Values
🔹 A. Replace Missing Customer_Name → "Unknown"
Select the Customer_Name column
Press Ctrl + H (Find & Replace)
In:
Find what: (leave blank)
Replace with: Unknown
Click Replace All

👉 Done — all empty names are now "Unknown"

🔹 B. Replace Missing Sales → Average
Step 1: Calculate Average Sales
Click an empty cell
Enter:
=AVERAGE(G2:G100)

(Adjust range based on your data)

Copy the result (Ctrl + C)
Step 2: Replace blanks
Select Sales column
Press Ctrl + G → Special → Blanks → OK
Type:
= (paste average value)
Press:
Ctrl + Enter

👉 All missing sales values are filled

🔹 C. Replace Missing Quantity → 1
Select Quantity column
Press Ctrl + G → Special → Blanks
Type:
1
Press Ctrl + Enter
✔️ 2. Format Data Properly
🔹 A. Convert Order_Date to Date Format
Select Order_Date column
Go to:
Home → Number Format dropdown
Choose:
Short Date or Long Date

👉 If dates look wrong:

Go to Data → Text to Columns → Finish
🔹 B. Ensure Numeric Columns

Check these columns:

Sales
Quantity
Discount
Fix:
Select column
Go to:
Home → Number Format → Number

👉 If numbers are left-aligned:

Click warning icon → Convert to Number
✔️ 3. Remove Duplicates
Select entire dataset (Ctrl + A)
Go to:
Data → Remove Duplicates
Click:
OK

👉 Excel will tell you how many duplicates were removed

✔️ 4. Create New Columns

Now we make your dataset “analysis-ready” 💪

🔹 A. Total Revenue Column
Add new column → name: Total Revenue
In first row enter:
=G2*H2*(1-I2)

(Adjust columns based on your layout)

👉 Example:

G = Sales
H = Quantity
I = Discount
Press Enter
Drag down
🔹 B. Month Column
Add new column → name: Month
Enter:
=TEXT(B2,"MMM")

👉 Converts date into:
Jan, Feb, Mar

🔹 C. Year Column
Add column → name: Year
Enter:
=YEAR(B2)

Before moving to Power BI:

✔ No blank cells
✔ All numbers formatted correctly
✔ Dates working properly
✔ New columns calculated
