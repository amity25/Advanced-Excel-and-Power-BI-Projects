DAX MEASURES (UPDATED WITH "Cleaned" TABLE)

🧠 What is DAX? (Very Simple Explanation)

👉 DAX (Data Analysis Expressions) = formulas used in Power BI
👉 Similar to Excel formulas, but works on entire tables + filters

🔑 Key Idea:
Column → row-by-row calculation
Measure → dynamic calculation (changes with filters)

👉 In real projects → we use Measures


🟢 Step 1: Where to Create Measure

Open Power BI
Go to Report View (chart icon on left)
On right side (Fields panel)

Find your table:

Cleaned

Right-click → Click:

New Measure

🖥️ What you’ll see:

A formula bar appears at the top like:

Measure =


🟢 Step 2: Create Total Revenue (MOST IMPORTANT)

📊 Formula:

Total Rev = 
SUMX(
    Cleaned,
    Cleaned[Sales] * Cleaned[Quantity] * (1 - Cleaned[Discount])
)
🧠 Explanation (Very Important)

🔹 SUMX means:

👉 “Go row by row and calculate, then add everything”

Inside SUMX:
Cleaned[Sales] * Cleaned[Quantity] * (1 - Cleaned[Discount])

👉 For each row:

Sales × Quantity
Apply Discount

Then:
👉 SUMX adds all rows → Total Rev

🖥️ What you’ll see:

A new field:

Total Revenue
With a calculator icon 🧮 (means measure)


🟢 Step 3: Create Total Sales

Total Sales = SUM(Cleaned[Sales])

🧠 Explanation:

SUM = simple addition of a column
No row-level calculation needed


🟢 Step 4: Create Total Quantity

Total Quantity = SUM(Cleaned[Quantity])


🟢 Step 5: Create Average Sales

Avg Sales = AVERAGE(Cleaned[Sales])

🧠 Explanation:

👉 Finds average value of Sales column


🟢 Step 6: Create Profit (Important)

📊 Formula:

Profit = 
SUMX(
    Cleaned,
    (Cleaned[Sales] * Cleaned[Quantity] * (1 - Cleaned[Discount])) * 0.2
)
🧠 Explanation:

We assume:

Profit = 20% of revenue

So:

Revenue × 0.2


🟢 Step 7: Profit Margin %

Profit Margin % = 
DIVIDE(
    [Profit],
    [Total Rev]
)
🧠 Why DIVIDE (not /)?

👉 DIVIDE() prevents errors when denominator = 0
👉 This is best practice

🎯 Format it:

Click measure
Go to Top menu → Format → Percentage (%)


🟢 Step 8: Time Intelligence (Advanced 🚀)

🔹 Revenue Last Year
Revenue LY = 
CALCULATE(
    [Total Rev],
    SAMEPERIODLASTYEAR(Cleaned[Order_Date])
)
🧠 Explanation:

CALCULATE → changes filter context
SAMEPERIODLASTYEAR → shifts date back 1 year

👉 Gives last year revenue

🔹 Growth %
Growth % = 
DIVIDE(
    [Total Rev] - [Revenue LY],
    [Revenue LY]
)

👉 Format as %
