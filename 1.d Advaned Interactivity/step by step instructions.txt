ADVANCED INTERACTIVITY

🔍 PART 1: DRILL-THROUGH (MOST IMPORTANT)

🧠 What is Drill-through?

👉 It lets you:

Right-click on a value (like Region)
Open a detailed page for that selection

Example:
👉 Click “North” → See all details for North only


🟢 Step 1: Create Drill-through Page

At bottom, click:

+ New Page

Rename it:

Region Details


🟢 Step 2: Add Drill-through Filter

Click empty space on new page

On right side, find:

Drill-through section

Drag: Region into it

✅ You’ll see:

“Add drill-through fields here”
→ Region added

👉 Now this page is linked to Region


🟢 Step 3: Add Visuals to Drill Page

Add these visuals:

📊 A. Table

Click Table visual
Add:
Customer_Name
Product
Total Revenue

📊 B. Bar Chart

Axis → Product
Values → Total Revenue

📊 C. Card

Total Revenue

👉 This page now shows detailed breakdown


🟢 Step 4: Add Back Button

Go to:

Insert → Buttons → Back
Place it on top

👉 This lets user go back to main dashboard


🟢 Step 5: Use Drill-through (TEST)
Go back to Main Dashboard

Right-click on:

Region (e.g., North)

Click:

Drill-through → Region Details

🎯 Result:

👉 New page opens showing ONLY North data

🎉 Drill-through complete!




🔘 PART 2: BUTTONS (NAVIGATION)

🧠 What are Buttons?

👉 Used for:

Navigation between pages
Making dashboard interactive


🟢 Step 6: Add Navigation Button
Go to main dashboard

Click:

Insert → Buttons → Blank


🟢 Step 7: Configure Button
Select button
Go to Format panel
Turn ON:
Action → On
Set:

Type:

Page Navigation

Destination:

Region Details


🟢 Step 8: Rename Button

Turn ON:

Text

Write:

View Region Details

👉 Now button is clickable


🟢 Step 9: Test Button

Hold:

Ctrl + Click
Click button

👉 It should take you to Region Details page


🟢 Step 10: Final User Flow

Now your dashboard works like this:


Main Dashboard
   ↓
Click Button OR Right-click Region
   ↓
Drill-through Page (Detailed View)
   ↓
Back Button
   ↓
Main Dashboard
