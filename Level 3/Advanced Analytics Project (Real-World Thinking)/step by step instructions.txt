🚀 STEP 2: Customer Segmentation (Core of Your Project)

Now we will convert raw numbers into business insights.

Right now you only have:

Total Sales
Total Profit
Order Count

👉 But companies don’t look at raw numbers.
They classify customers into groups like:

🟢 High Value
🟡 Medium Value
🔴 Low Value


🧠 STEP 2.1: Add a New Column

In your Customer Summary table, add a new column:

👉 Name it: Customer Segment


🪜 STEP 2.2: Define Segmentation Logic

We will use Total Sales (simple & beginner-friendly).

👉 Rule:
If Sales ≥ 2000 → High Value
If Sales between 1000–1999 → Medium Value
If Sales < 1000 → Low Value


⚙️ STEP 2.3: Apply Formula in Excel

Assuming:

Total Sales is in column B
First data row is row 2
👉 Use this formula:
=IF(B2>=2000,"High Value",IF(B2>=1000,"Medium Value","Low Value"))
👉 Drag the formula down

Now every customer will automatically get a segment.

🪜 STEP 3.3: Create Customer Segment (DAX)

Now the powerful part.

👉 Create New Column
Go to Data View
Click New Column
👉 Paste this DAX:

Customer Segment = 

IF(
    'Sheet1'[Sales] >= 2000, "High Value",
    IF('Sheet1'[Sales] >= 1000, "Medium Value", "Low Value")
)
🧠 But Wait (Real Analyst Thinking)

The above is row-level segmentation ❌
We want customer-level segmentation ✅

👉 So we will fix this in next step (don’t worry)


🪜 STEP 3.4: Create Proper Measure (Correct Way)

Click New Measure and use:

Total Sales = SUM('Sheet1'[Sales])

Now:

Customer Segment Measure = 
SWITCH(
    TRUE(),
    [Total Sales] >= 2000, "High Value",
    [Total Sales] >= 1000, "Medium Value",
    "Low Value"
)
